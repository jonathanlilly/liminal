# Change Log

October 27, 2022

Fixed inconsistent sizing of KaTeX between inline and displayed equations, and added a style in the header to control the equation font size.

Updated to the following
v0.16.3 of [KaTeX][1], including katex.min.js and [auto-render.min.js][2] and [katex.min.css][3] and fonts directory
v0.15.0 of [remark-latest.min.js][4]

Note that you have to grab the minimized files from the CDN, you can't use the GitHub repositories.
